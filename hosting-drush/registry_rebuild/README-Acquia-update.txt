This directory was added from the root dir using:

git subtree add --squash --prefix=hosting-drush/registry_rebuild http://git.drupal.org/project/registry_rebuild.git 7.x-1.7

so it started from the 7.x-1.7 release code.

To update to the latest e.g on the 7.x-1.x branch do:

git subtree pull --prefix=hosting-drush/registry_rebuild http://git.drupal.org/project/registry_rebuild.git 7.x-1.x
